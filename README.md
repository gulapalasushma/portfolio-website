# 🌐 Personal Portfolio Website

A modern and responsive **Personal Portfolio Website** built using **HTML, CSS, and JavaScript**. It showcases my profile, skills, education, projects, certificates, and contact information with a 3D animated user interface.

## ✨ Features

* 👤 About Me
* 💻 Skills
* 🎓 Education Roadmap
* 📂 GitHub Projects
* 📜 Certificates
* 🔗 GitHub & LinkedIn Links
* 📧 Contact Form
* 🎨 3D Animated Background
* 📱 Responsive Design

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript
